# PEDDI — Cinematic Promotional Website

A high-fidelity, cinematic promotional website for the Telugu film **PEDDI** starring Ram Charan.
Built dynamically with React, Vite, and Tailwind CSS.

## 🚀 Features
- **Dynamic Language Support:** Animated title transitions for pan-India audiences.
- **Cinematic Themes:** Premium UI with a tailored gold (`#D4AF37`), pure black, and rust color palette.
- **Performance Optimized:** Lazy loading intersection observers for stats, purely CSS-driven loading phases, and responsive waveform tracking.
- **Interactive UI Components:** Custom-built React lightbox galleries, sticky category-based news filtering, and auto-playing track visuals.

## 🛠 Tech Stack
- **Framework:** React 18 + Vite
- **Styling:** Tailwind CSS v3
- **Icons:** Lucide React & inline custom SVG assets
- **Routing:** React Router DOM

## 📦 Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server (Defaults to `localhost:5173`):
   ```bash
   npm run dev
   ```

## 🌐 Build

To spin up a production-ready build:
```bash
npm run build
```
Generates statically optimized files inside the `dist/` directory.
